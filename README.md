# 🩺 Health Guardian AI

Health Guardian AI is a full-stack healthcare management web application that helps users manage their health records, medicine reminders, family health, prescriptions, and AI-powered health assistance.

## 🚀 Features

- 🔐 User Authentication
- 👨‍👩‍👧 Family Health Management
- 💊 Medicine Reminder
- 📅 Health Checkup Reminder
- 🤖 AI Health Chatbot
- 📄 Prescription Scanner
- 👤 User Profile Management

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- Multer

### Database
- PostgreSQL

## 📂 Project Structure

```text
Health-Guardian-AI/
│
├── frontend/
├── backend/
└── database/
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Nisha8806/Health-Guardian-AI.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Database

Import the `database/schema.sql` file into PostgreSQL.
